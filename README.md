# catcatch-backend

https://github.com/pongdang/catcatch

- oauth 기능 (구글, 페이스북, 네이버)
- 사진 업로드 기능 (public)
- 해시태그 기반 검색엔진
- 기본적인 MySQL 데이터 사용 ( USER, OAUTH, PHOTO, POST, TAG ... )

- Terraform with AWS 자동화

  - S3 연동
  - MySQL 연동
  - nestjs 서버

- 도메인 예정
  - 프론트: catcatch.co.kr
  - 백엔드/기타: catcatch-service.co.kr
